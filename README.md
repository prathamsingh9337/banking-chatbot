Banking Support Chatbot
An AI-powered banking support chatbot built with RAG (Retrieval-Augmented Generation).
Tech Stack

Backend: Node.js + Express.js
RAG: LangChain.js
Vector DB: ChromaDB
Embeddings: HuggingFace all-MiniLM-L6-v2 (free, local)
LLM: Groq API (LLaMA3 - free)
Frontend: Plain HTML + CSS + JavaScript
